# Jr Collection — Catálogo de Discos de Vinil

App pessoal para catalogar sua coleção de discos, com scanner de capa por IA.

---

## Como colocar no ar (passo a passo)

### 1. Criar conta no GitHub
- Acesse https://github.com e crie uma conta gratuita

### 2. Criar repositório
- Clique em **New repository**
- Nome: `jr-collection`
- Deixe como **Public**
- Clique em **Create repository**

### 3. Fazer upload dos arquivos
- Na página do repositório, clique em **uploading an existing file**
- Arraste a pasta descompactada (todos os arquivos)
- Clique em **Commit changes**

### 4. Criar conta no Vercel
- Acesse https://vercel.com
- Clique em **Sign up** → **Continue with GitHub**
- Autorize o Vercel

### 5. Publicar o app
- No Vercel, clique em **Add New Project**
- Selecione o repositório `jr-collection`
- Clique em **Deploy**
- Aguarde ~2 minutos

### 6. Seu app está no ar!
- O Vercel vai gerar um link tipo: `jr-collection.vercel.app`
- Abra esse link no celular

---

## Instalar no celular (Android)

1. Abra o link no **Google Chrome**
2. Toque nos **3 pontinhos** (menu) no canto superior direito
3. Toque em **"Adicionar à tela inicial"**
4. Confirme — o ícone aparece na sua tela!

Agora abre igual qualquer app, sem barra de navegador.

---

## Funcionalidades

- 🔍 **Busca inteligente** — por artista, disco, música ou gravadora
- 📷 **Scanner de capa** — aponta a câmera e a IA identifica o disco
- 🟢🟡🔴 **Status de lavagem** — semáforo de quando lavar
- 💧 **Registrar lavagem** — botão "Lavei agora"
- 💾 **Dados salvos localmente** — não perde nada ao fechar
- 📱 **PWA** — instala como app no celular
